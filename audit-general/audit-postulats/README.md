# Pilote des postulats 2021–2026

Cette première phase couvre les 35 postulats officiels : nouveau scraper JSON,
métadonnées documentaires de base, métadonnées additionnelles et audit de
redondance. L'audit général des textes, données manquantes et chunks sera fait
dans une seconde phase.

Ouvrir `pilot/review.html` pour consulter le résultat.
